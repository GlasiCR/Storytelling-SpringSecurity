## Funcionalidades:
- Cadastro de usuário;
- Login com validação de senha e token (JWT); 
- Atualização de dados do usuário;
- Cadastro de Hotel;
- Cadastro de Quarto e relacionamento com o Hotel;
- Filtro por nome do hotel e cidade.
